# Data Science Task - Ankan_Pal

## Project Overview
This project analyzes trading performance correlation with the Fear & Greed Index.

## Folder Structure
- `notebook_1.ipynb`: The Python code used for analysis.
- `ds_report.pdf`: A summary PDF report containing insights and charts.
- `outputs/`: Folder containing the generated .png charts.
- `csv_files/`: Folder containing the raw data used.

## Instructions
1. Open `notebook_1.ipynb` in Jupyter Lab or Google Colab.
2. Ensure `csv_files/` contains the dataset.
3. Run all cells to reproduce the analysis.
